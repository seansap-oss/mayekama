# MayekAma iOS Keyboard Scaffold

Apple custom keyboards must be built as an iOS app plus a Custom Keyboard extension target in Xcode.

Use this folder as source scaffold:

- `HostApp/MayekAmaApp.swift` — simple settings host app.
- `KeyboardExtension/KeyboardViewController.swift` — custom keyboard extension view controller.
- `KeyboardExtension/Info.plist` — extension configuration sample.

Create an Xcode iOS app, add a Custom Keyboard Extension target, then copy these files into the matching targets.
