# MayekAma condensed 6-build plan

## Build 1 — Foundation and approved UI
Website shell, design direction, language-engine base, Supabase schema, platform folders.

## Build 2 — Public website and writer
Landing page, writer, dictionary, contribution queue, standard notes, AI draft workspace.

## Build 3 — Keyboard foundation
Native Android IME scaffold, companion app, local language pack, suggestion-strip logic, iOS notes.

## Build 4 — Bigger integration build
This package combines the previous 12-stage plan into a larger production foundation:

- Website + PWA
- Writer + dictionary + contribution workflow
- Language engine candidates for keyboard use
- Supabase database and API planning
- Android keyboard alpha source
- Companion app source
- One-command Windows setup workflow
- Doctor/status scripts

## Build 5 — APK and platform testing build
Focus: Android Studio sync, debug APK build, phone install, enable/select keyboard, WhatsApp/Facebook/YouTube typing tests, keyboard UI polish, local storage, settings.

## Build 6 — Release candidate
Focus: production polish, SEO, deployment, admin hardening, API keys, Supabase integration, docs, APK/AAB signing path, and handover.

Current package: **Build 4 of 6**.
