# MayekAma PowerShell Commands

## Command 1 – Install website dependencies

```powershell
npm install
```

## Command 2 – Run website

```powershell
npm run dev
```

## Command 3 – Open website

```powershell
start http://localhost:5173
```

## Command 4 – Test language engine

```powershell
npm run test:engine
```

## Command 5 – Validate local language pack

```powershell
npm run validate:pack
```

## Command 6 – Open Android keyboard folder

```powershell
start "" "D:\MayekAma\apps\android-keyboard"
```
