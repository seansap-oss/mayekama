# UI Lock Rule

MayekAma visual direction is locked from Build 5 onward.

## Rule
- Do not redesign approved sections.
- Do not remove existing visible sections.
- Only add new capability, documentation, scripts, API endpoints, database tables, keyboard behavior, and production hardening.
- If a technical fix requires visible UI change, document it in CHANGELOG before changing it.

## Approved direction
Cream / maroon / editorial / culturally respectful / simple / mobile friendly.
