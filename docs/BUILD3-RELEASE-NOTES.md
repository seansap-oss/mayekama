# MayekAma Build 3 Release Notes

## Added

- Expanded starter dictionary seed from 15 to 40 entries.
- Added domains for daily chat, school, writing, technology, places and family terms.
- Added local language pack export/validation functions.
- Added `getKeyboardCandidates()` API for Android/iOS keyboard usage.
- Added native Android Gradle project scaffold.
- Added Android companion activity with enable/select flow.
- Added Android `InputMethodService` keyboard scaffold.
- Added Android local `LanguagePack.kt` starter candidates.
- Added Android test notes.

## Not final yet

- The Roman Manipuri word list is still a seed list, not an authoritative complete dictionary.
- The Android keyboard is a functional scaffold, not final UI polish.
- iOS still remains source notes only; native iOS build needs Xcode on Mac.
